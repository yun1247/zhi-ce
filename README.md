# 🧭 职测 · 你的职业匹配助手

> 把你的性格、技能、偏好告诉我，我来帮你找到最适合的职业方向。

## 这是什么

职测是一个开源的性格-职业匹配工具。不同于传统职业测评只给笼统建议，职测通过 **多维匹配算法**，把你的性格、兴趣、技能、工作偏好与真实岗位进行量化匹配，输出**匹配度百分比**和详细分析。

### 核心特色

- **🎯 多维度匹配**：性格 × 兴趣 × 技能 × 工作方式 × 生活方式 × 价值观
- **📊 量化结果**：每个职业给出 0-100% 匹配度，附带各维度得分
- **🔍 透明算法**：匹配逻辑完全开源，你可以看到为什么推荐这个职业
- **📱 开箱即用**：纯前端，不需要任何后端，浏览器直接打开
- **🔌 可扩展**：任何人都可以提交 PR 添加更多职业数据

- 纯前端：HTML5 + CSS3 + Vanilla JS
- 数据格式：JSON（`data/careers.json`）
- 匹配引擎：`engine/matcher.js`（~150 行核心算法）
- 总大小：< 100KB

## 匹配算法说明

基于**霍兰德六型人格理论** + **关键字匹配**的加权综合算法：

| 维度 | 权重 | 匹配方法 |
|------|------|----------|
| 性格 (RIASEC) | 25% | 六维差异余弦相似度 |
| 兴趣 | 20% | 关键词重叠率 |
| 技能 | 20% | 技能覆盖度 |
| 工作方式 | 15% | 偏好匹配 |
| 生活方式 | 10% | 兼容度 |
| 价值观 | 10% | 一致性 |
